# Assets Directory

This directory contains static assets used by the backend services.

## Logo

Place your organization logo here as `logo.png` to include it in PDF documents.

**Requirements:**
- Filename: `logo.png`
- Format: PNG (recommended) or JPG
- Recommended size: 300x300 pixels or similar square aspect ratio
- Maximum width in PDF: 90 pixels

**Usage:**
The logo will be automatically embedded in:
- Ward Attendance Register PDFs
- Other PDF documents generated by the system

**Example:**
```
backend/assets/logo.png
```

If no logo file is found, PDF documents will be generated without a logo.
