## Features

1. **Header Bar**
   - Organization logo
   - System name
   - Notification bell icon with counter
   - Profile dropdown menu

2. **Navigation Sidebar**
   - Dashboard (current page)
   - Profile management
   - Membership details
   - Documents
   - Renewal
   - Notifications

3. **Status Cards**
   - Membership status with expiration date
   - Voter registration status with verification date
   - Document summary with quick access

4. **Recent Activities**
   - Timeline of recent member activities
   - Date and description of each activity
   - Link to view all activities

5. **Announcements**
   - Upcoming events and system announcements
   - Date and brief description
   - Relevant to member's ward and hierarchy

## Interactions

- Clicking on status cards navigates to detailed views
- "View All" links expand to show complete lists
- Notification bell opens notification dropdown
- Profile icon opens user menu with logout option
