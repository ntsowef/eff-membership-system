## Design Guidelines
- All screens follow responsive design principles
- Color scheme: Professional palette (blue, white, gray)
- Typography: Clean, readable fonts (Roboto or Open Sans)
- Icons: Material Icons or FontAwesome
- Consistent navigation and layout across all screens
